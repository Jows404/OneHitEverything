Jows404's OneHitEverything Plugin

OneHitEverything is a lightweight, no-dependency Minecraft plugin that sets every living entity in the game, players, mobs, 
and bosses to exactly 1 heart of health. One hit equals instant death.

## Features
- **One Hit Kill** ‚Äì Every entity dies in one hit. Players, mobs, bosses. No exceptions.
- **Boss Compatible** ‚Äì Ender Dragon, Wither, Elder Guardians. All 1 HP.
- **Toggleable** ‚Äì Enable/disable anytime with `/onehit enable` or `/onehit disable`
- **Permission Support** ‚Äì Only admins can use the commands
- **Lightweight** ‚Äì Zero dependencies, no performance 
- **Custom Death Message** ‚Äì Configurable message when players die

## Commands
| Command           | Description        | Permission |
|-------------------|--------------------|------------|
| `/onehit help`    | List of commands   | plugin.admin |
| `/onehit enable`  | Enables challenge  | plugin.admin |
| `/onehit disable` | Disables challenge | plugin.admin |
| `/onehit reload`  | Reload config      | plugin.admin |
